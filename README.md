# ptrfind-windbg
Ptrfind for Windbg.
Port of [ptrfind](https://github.com/ChaChaNop-Slide/ptrfind) to Windows.

# Load Script
In Windbg: .scriptload <path-to-ptrfind.js>

# Print Usage:
!ptrfind
